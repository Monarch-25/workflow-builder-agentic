## Rules

The following rules apply to the entire project:

1. you cannot execute any code that relies on calling some external service (bedrock, redis, etc.).
2. This repo is only to create code it has no necessary auths to execute any code.
3. You can only check the syntax of the code, but you cannot execute it.
4. If you need any logs for the code which requires auths then give the user that code in a file and ask them to execute it.
5. Be very clear about the plan of action and if there are deviations or suggestions then inform the user.
